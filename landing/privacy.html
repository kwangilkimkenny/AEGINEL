Aegis Personal

Privacy Policy

Last updated: March 9, 2026

TL;DR — Aegis Personal does not collect, store, or transmit any of your data. Everything runs locally in your browser.

1. Overview

Aegis Personal ("AI Safety Guard") is a Chrome extension developed by YATAV Inc. (Seoul, Korea). This privacy policy explains how Aegis Personal handles user data.

2. Data We Do NOT Collect

Aegis Personal is designed with a zero data collection architecture. We do not collect, store, or transmit:

  • Your prompts or messages typed into AI services
  • Personal information (PII) detected by the extension
  • Browsing history or website content
  • Usage analytics or telemetry
  • IP addresses or device identifiers
  • Cookies or tracking data

3. How Aegis Personal Works

All processing happens entirely within your browser:

  • Threat detection — Rule-based and ML-based analysis runs locally using JavaScript and WebAssembly (ONNX Runtime)
  • PII detection & masking — Personal data is detected and pseudonymized in-browser before being sent to any AI service
  • Settings storage — Your preferences are saved using Chrome's local storage API (chrome.storage.local), which stays on your device
  • ML model — The AI Guard model runs in a Chrome Offscreen Document using Transformers.js. No data is sent to any external API for classification

4. Permissions Explained

Aegis Personal requests the following Chrome permissions:

  • activeTab — To detect and protect input fields on the currently active AI service tab
  • storage — To save your settings locally on your device
  • offscreen — To run the ML classification model in a background document (required for WebAssembly execution in Manifest V3)
  • Host permissions — To inject content scripts into supported AI service websites (ChatGPT, Claude, Gemini, etc.) for real-time protection

5. Third-Party Services

Aegis Personal does not integrate with any third-party analytics, advertising, or data processing services. No data is shared with third parties.

6. Data Retention

Since no data is collected, there is no data to retain. Scan statistics shown in the popup dashboard are stored locally via chrome.storage.local and can be cleared at any time from the settings panel.

7. Children's Privacy

Aegis Personal does not knowingly collect any personal information from anyone, including children under 13.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date. Continued use of Aegis Personal after changes constitutes acceptance of the updated policy.

1. 개요

Aegis Personal("AI Safety Guard")은 주식회사 야타브(서울, 대한민국)가 개발한 Chrome 확장 프로그램입니다. 본 개인정보 처리방침은 Aegis Personal이 사용자 데이터를 어떻게 처리하는지 설명합니다.

2. 수집하지 않는 정보

Aegis Personal은 데이터 무수집(Zero Data Collection) 아키텍처로 설계되었습니다. 다음 정보를 수집, 저장, 전송하지 않습니다:

  • AI 서비스에 입력하는 프롬프트 또는 메시지
  • 확장 프로그램이 탐지한 개인정보(PII)
  • 브라우징 기록 또는 웹사이트 콘텐츠
  • 사용 분석 또는 텔레메트리 데이터
  • IP 주소 또는 기기 식별자
  • 쿠키 또는 추적 데이터

3. Aegis Personal의 작동 방식

모든 처리는 사용자의 브라우저 내에서 이루어집니다:

  • 위협 탐지 — 규칙 기반 및 ML 기반 분석이 JavaScript와 WebAssembly(ONNX Runtime)를 통해 로컬에서 실행됩니다
  • PII 탐지 및 가명화 — 개인정보가 브라우저 내에서 탐지되고 AI 서비스로 전송되기 전에 가명화됩니다
  • 설정 저장 — 사용자 설정은 Chrome 로컬 스토리지 API(chrome.storage.local)를 통해 기기에만 저장됩니다
  • ML 모델 — AI Guard 모델은 Chrome Offscreen Document에서 Transformers.js를 통해 실행됩니다. 분류를 위해 외부 API로 데이터를 전송하지 않습니다

4. 권한 설명

Aegis Personal은 다음 Chrome 권한을 요청합니다:

  • activeTab — 현재 활성 AI 서비스 탭의 입력 필드를 감지하고 보호하기 위함
  • storage — 사용자 설정을 기기에 로컬로 저장하기 위함
  • offscreen — 백그라운드 문서에서 ML 분류 모델을 실행하기 위함 (Manifest V3에서 WebAssembly 실행에 필요)
  • 호스트 권한 — 지원되는 AI 서비스 웹사이트(ChatGPT, Claude, Gemini 등)에 콘텐츠 스크립트를 주입하여 실시간 보호를 제공하기 위함

5. 제3자 서비스

Aegis Personal은 제3자 분석, 광고 또는 데이터 처리 서비스와 연동하지 않습니다. 제3자와 데이터를 공유하지 않습니다.

6. 데이터 보존

데이터를 수집하지 않으므로 보존할 데이터가 없습니다. 팝업 대시보드에 표시되는 스캔 통계는 chrome.storage.local을 통해 로컬에 저장되며, 설정 패널에서 언제든지 삭제할 수 있습니다.

7. 아동 개인정보

Aegis Personal은 13세 미만 아동을 포함한 누구의 개인정보도 의도적으로 수집하지 않습니다.
